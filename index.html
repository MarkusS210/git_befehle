Initialen Upload für neue Projekte

Startet man ein neues Projekt, welches auf Github hochgeladen werden soll, so nutzt man die initialen Upload Befehle.

Befehle für neue Projekte

  • git init - Ein neues Git-Respository wird erstellt
  • git add . - Erstellt ein neuen commit, welches auf Github geladen wird
  • git commend -m "Name" - Erstellt einem Namen für das neue commit

    • Upload für bestehendem Projekt

    Besteht auf Github bereits eine Repository, so kann man mit den Uploadbefehlen ein neun commit dazu erstellen und aktualisieren

    Befehle für bestehende Projekte

  • git add . - Erstellt ein neuen commit, welches auf Github geladen wird
  • git commend -m "Name" - Erstellt einem Namen für das neue commit
  • git push - wird verwendet, um Inhalte aus einem lokalen Repository in ein Remote-Repository hochzuladen.
